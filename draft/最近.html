最近

六号从北京回家，正打算整理最近笔记。可惜人太懒，很多只是开了小头，没有完整写下去。思来想去，还是觉得必须强迫自己先写点东西发出来。[1]

这几天最大的体悟还是有想法就要立马去做，很开心的是自己很多想做的东西确实实现了。去翻了先前零散的几篇日记，发现写博客的想法大概是在三月产生的，在七月份就实现了，我确实没有想到会有这么快。

简单总结一下最近：

  • 学习了 Docker。非常好的机会，跟 git 一样很实用的工具，一直想学，正好遇到 issue 可以顺带 learn by practicing。

  • 重拾 Fl studio。水果机在高中就玩过，大学后接触到了 Cubase，但还是水果机的界面用起来舒服（平滑动效，使用体验极佳）。

    • 借七夕的 ddl 仓促学了混音（压缩 + eq），效果不错！而且发现写歌词很难，可能就是因为自己太久没有进行过写作了，在写作想象力方面已经是一条岸上濒临窒息的鱼了。[2]
    • 最近到了声卡和麦克风，测试了一下发现远超我的手机麦克风。终于有机会可以真正听到自己的声音是什么样子，不用忍受嘈杂的电流声，把录音多学学，说不定真能成为卧室音乐人。

  • 简单接触 Rust & 编译原理。

    • Rust 似乎没有想象中的如此有趣，&mut self的写法感觉非常的无聊，'a的写法简直也丑到了极致，更不用提结构体的几种“茴”字写法，.0，.1简直丑的令人呕吐。相比之下，c++ 的写法算优雅的了（真就比烂上了）。学习体验远不如 Haskell（虽然又是一个没有填的坑，但是 Haskell 学起来真的趣味十足）。
    • 编译原理自从学校回来后就没有进展了。如果效率高一点，说不定能写个 demo[3]。不过接触到了 cmake，也算是学有所获。关于 cpp 的项目结构还需要再查查，原生自带的还是太过于落后了，在这一点 Rust 确实做的好，Cargo 非常的香😋！

  • 健身稍微的提上了日程。发现多接触基本还是能度过小白的阶段，而且自己本身也有底子在。可惜配置录音设备+换琴弦耗费财力略多，到学校后能否办卡还是比较焦虑，更何况十月还有旅游计划，希望老师能多发点工资，实在不行去做点家教。

接下来的想法：

  • 音乐继续做。很可能到学校后在宿舍没有在家里如此自由的环境，设备会蒙灰。趁现在还不太忙，一定要抓紧时间学，抓紧时间做！

  • 学习英语。最后一次六级的机会，一定要抓住。好好学英语也是一直以来的想法，甚至可以比如英语博客写作，应该很能锻炼英语水平。

  • 健身坚持下去。正好也是体测，虽然没有体育课了，但是希望自己的引体真能突破。

  • 解释器。还是想写出来，也很可能现在不写以后也没有机会了。

总体来说过得不错。如果玩手机和熬夜的时间再短一点就更好了，不熬夜能使我的皮肤变好。

  1. 这篇博客写的既流水又烂，我自己都看不下去，抽时间一定再修改修改！ ↩︎

  2. 必须承认的是我有很强的炫耀欲，虽然发到朋友圈后没有达到我想要的反响。 ↩︎

  3. 更有可能是我写到一半就放弃了。 ↩︎

Let no one ignorant of thinking enter.
