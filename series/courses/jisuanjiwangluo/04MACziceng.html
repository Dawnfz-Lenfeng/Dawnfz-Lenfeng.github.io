2.1.2 CSMA

Definition (非持续式 CSMA)

  1. 经侦听，如果介质空闲，开始发送
  2. 如果介质忙，则等待一个随机分布时间，再重复步骤 1

Definition (1-持续式 CSMA)

  1. 经侦听，如果介质空闲，则发送
  2. 如果介质忙，持续侦听，一旦空闲立即发送
  3. 如果发送冲突，等待一个随机分布时间再重复步骤 1

Definition (p-持续式 CSMA)

  1. 经侦听，如果介质空闲，以概率发送，以概率延迟一个时间单元发送
  2. 如果介质忙，持续侦听，一旦空闲重复 1
  3. 如果发送推迟一个时间单元，重复步骤 1

Definition (1-持续式 CSMA/CD)

  1. 经侦听，如果介质空闲，则发送
  2. 如果介质忙，持续侦听，一旦空闲立即发送
  3. 如果发生冲突，立即中止自己的传输，等待随机分布时间再重复步骤 1

Definition (令牌传递)

只有持有令牌的站点才能发送数据。令牌轮流传递。

Definition (二进制倒计数协议)

每个站点编序号，序号长度相同。在竞争期有数据发送的站点从高序号到低序号排队，高者得到发送权。

2.3 有限竞争

Definition (自适应数搜索协议)

所有节点构成完全二叉树。在竞争时隙，所有站点同时竞争。如果只有一个站点申请，则获得信道。否则下一竞争时隙，一半站点参与竞争，另一半时隙参与下一时隙竞争

3. 以太网

Definition (MAC 地址)

6 字节的全局唯一的设备标识，又称为硬件地址或物理地址。

Definition (集线器)

集线器是一种物理层设备，用于在多个计算机之间进行电信号的广播传输。它将一个端口接收到的数据原封不动地广播到所有其他端口，不区分发送目标，也无法识别数据内容。

Definition (交换机)

交换机是一种数据链路层设备，用于根据数据帧中的MAC 地址进行转发决策。它能学习各个端口对应的 MAC 地址，并只把数据帧转发给目标端口，实现点对点通信

Definition (以太网)

以太网是一种局域网通信技术标准，由 IEEE 802.3 定义，规定了数据在局域网中以帧的形式进行传输的方式。它采用MAC 地址寻址，使用双绞线或光纤作为传输介质，支持的标准速率包括 10Mbps、100Mbps、1Gbps 及以上。以太网工作在OSI 模型的物理层和数据链路层，典型接入控制方式为CSMA/CD（在使用集线器时），而现代以太网多采用交换机实现全双工通信，无冲突，高效率，广泛应用于企业、家庭、校园等局域网环境中。

4. 数据链路层交换

4.1 交换模式

  • 存储转发：转发前接受整个帧，执行 CRC 校验
  • 直通交换：一旦接收到帧的目的地址，就开始转发
  • 无碎片交换：接收到帧的前 64 字节，即开始转发

4.2 生成树协议

Definition (生成树协议)

  1. 选举根桥：同一广播域中的所有交换机均参与选举，桥 ID 最小的交换机（网桥）成为生成树的根，其余为非根桥
  2. 为每个非根桥选出一个根端口：每个非根桥通过比较其每个端口到根桥的根路径开销，选出根端口，具有最小根路径开销的端口被选作根端口
  3. 为每个网段确定一个指定端口：对于每一个网段， 在所有连接到它的交换机（网桥）端口中进行选择，一个具有最小根路径开销的端口，作为该网段的指定端口

5. 无线局域网

5.1 架构体系

  • LLC层：逻辑链路控制层，向上（网络层）提供统一的接口
  • MAC层：OSI第2层（数据链路层）的另一部分，添加和识别MAC地址（硬件地址），用于局域网中节点之间识别
  • 物理层（PLCP/PMD层）

Definition (CSMA/CA)

  1. 经侦听，如果介质空闲，等 IFS，若仍空闲则发送
  2. 如果介质忙，持续侦听，一旦空闲立即发送
  3. 在发送前等待 IFS，若遇到不空闲，进行二进制指数退后再传输
