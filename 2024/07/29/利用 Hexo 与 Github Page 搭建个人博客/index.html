3.2 Markdown 引擎优化

Hexo 默认搭配的 Markdown 渲染器是hexo-renderer-marked，但其支持的渲染格式有限，因此更推荐使用功能更加强大，渲染速度更快的hexo-renderer-markdown-it渲染器3。经过测试，发现hexo-renderer-pandoc更好用，完美支持多行公式4！

3.3 Obsidian 工具链

关于写笔记的工具，这里推荐 Obsidian，能够很好的与 Hexo 进行联动兼容。

首先去Obsidian 官网下载 Obsidian，打开后在source目录下创建 Obsidian 仓库。此时我们可以直接在_posts下创建笔记，完美兼容 Obsidian 生态。
